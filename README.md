# Quicktasker

A task marketplace platform connecting clients with freelancers — Frontend built with Next.js, TypeScript, and Tailwind CSS.

## Overview

Quicktasker was a full-stack platform designed to help job posters find skilled freelancers for various tasks. As the **sole frontend developer**, I owned 85% of the project's codebase — architecting, building, and deploying the complete user interface while collaborating with a backend developer on API integration.



## Tech Stack

- **Next.js 15** — React framework with App Router
- **React 19** — UI library
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling and responsive design
- **Vercel** — Hosting and deployment

## Getting Started

### Prerequisites

- Node.js 18+ or 20+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Echika/quicktasker-frontend.git
cd quicktasker-frontend

# Install dependencies
npm install

# Run development server
npm run dev



npm run build
npm start
